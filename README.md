# Task 15: Full Stack Integration (MERN)

## Objective

Integrate React frontend with Express, Node.js, and MongoDB backend.

## Features

* Add Student
* View Students
* REST API Integration
* MongoDB Storage
* React UI
* CORS Handling

## Technologies

* MongoDB
* Express.js
* React.js
* Node.js
* Axios
* CORS

## Workflow

React → Axios → Express API → MongoDB

## Outcome

Successfully connected frontend, backend, and database into a complete MERN application.
